### Codeblock

Naprawione formatowanie

```tsx
function test() {
	console.log("test");
}
```

Można dodać linie numerowane

```js showLineNumbers
const file = await remark().use(remarkGfm, { stringLength: stringWidth });

function test() {
	console.log("test");
}
```

```js showLineNumbers{99}
const file = await remark().use(remarkGfm, { stringLength: stringWidth });

function test() {
	console.log("test");
}
```

Podświetlić konkretne linie


```js showLineNumbers {1,3-5}
const file = await remark().use(remarkGfm, { stringLength: stringWidth });

function test() {
	console.log("test");
}
```

Podświetlić diffa

```js {1}#a {3-5}#d
const file = await remark().use(remarkGfm, { stringLength: stringWidth });

function test() {
	console.log("test");
}
```

Podświetlić konkretny fragment

```js /test/
const file = await remark().use(remarkGfm, { stringLength: stringWidth });

function test() {
	console.log("test");
}
```

```js /test/2
const file = await remark().use(remarkGfm, { stringLength: stringWidth });

function test() {
	console.log("test");
}
```

```js /console/2,4-5 /test/1,5-7
console.log("test");
console.log("test");
console.log("test");
console.log("test");
console.log("test");
console.log("test");
console.log("test");
```
