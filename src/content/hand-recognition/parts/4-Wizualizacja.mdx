Mamy surowe współrzędne 21 punktów, ale dla człowieka to tylko lista liczb. Musimy je zwizualizować na żywo, nakładając cyfrowy szkielet na obraz z kamery. Wykorzystamy do tego element **HTML5 Canvas**.

### Topologia dłoni

Aby narysować dłoń, musimy wiedzieć, które punkty połączyć liniami. Definiujemy mapę połączeń:

```ts
const fingerJoints = [
  [0, 1], [1, 2], [2, 3], [3, 4],    // Kciuk
  [0, 5], [5, 6], [6, 7], [7, 8],    // Wskazujący
  [0, 9], [9, 10], [10, 11], [11, 12], // Środkowy
  // ... i tak dalej
];
```
Punkt `0` to nadgarstek, od którego rozchodzą się wszystkie palce.

### Potok rysowania

W każdej klatce wykonujemy trzy kroki:

1.  **Czyszczenie (`clearRect`)**: Musimy wymazać poprzednią dłoń. Bez tego na ekranie powstałyby "smugi" i nakładające się na siebie klatki.
2.  **Rysowanie linii**: Przechodzimy przez mapę połączeń i rysujemy kreski między stawami.
3.  **Rysowanie punktów**: Na koniec zaznaczamy każdy staw małym kółkiem (aby linie nie "wystawały" poza stawy).

```ts showLineNumbers
const drawHands = (hands, canvas) => {
  const ctx = canvas.getContext("2d");
  ctx.clearRect(0, 0, canvas.width, canvas.height); // Obowiązkowe!

  hands.forEach(hand => {
    ctx.strokeStyle = "white";
    ctx.fillStyle = hand.handedness === "Right" ? "blue" : "red";
    
    // Rysowanie struktury (linii)
    fingerJoints.forEach(([i, j]) => {
      ctx.beginPath();
      ctx.moveTo(hand.keypoints[i].x, hand.keypoints[i].y);
      ctx.lineTo(hand.keypoints[j].x, hand.keypoints[j].y);
      ctx.stroke();
    });
  });
};
```

> [!TIP]
> Zauważ, że rysujemy dłoń "od dołu do góry": najpierw linie, potem kropki. Dzięki temu kropki ładnie przykrywają punkty styku linii, tworząc profesjonalny wygląd szkieletu.

### Wyzwanie: Lustrzane odbicie

Ponieważ używamy kamery w trybie "selfie", obraz jest zazwyczaj odbity lustrzanie w CSS (`scaleX(-1)`). Musimy o tym pamiętać przy rysowaniu napisów lub asymetrycznych elementów, aby nie zostały one "odwrócone" dla użytkownika.
